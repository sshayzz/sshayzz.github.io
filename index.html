Space Invaders!

A Python-based recreation of the classic arcade game Space Invaders made in collaboration with Averi Wylie.

🎮 Features

  • Player-controlled spaceship with 8D movement
  • Enemy invaders that you must avoid
  • Projectile firing mechanics
  • Score tracking
  • Power-ups

🚀 Getting Started

Prerequisites

  • Python 3.11
  • Pygame library

Installation

git clone https://github.com/sshayzz/Project-1---Space-Invaders.git
 cd Project-1---Space-Invaders  
 pip install pygame

📁 Project Structure

Project-1---Space-Invaders/ ├── assets/ # Game graphics and sounds ├── application.py # Main game loop ├── interface.py # UI classes ├── character.py # Player and enemy classes ├── constants.py # Game constants and configurations ├── game_token.py # Token management ├── LICENSE # Project license ├── README.md # Project documentation └── index.html # Github pages website
